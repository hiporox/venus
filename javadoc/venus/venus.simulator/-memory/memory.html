memory

val memory: <ERROR CLASS>

A hashmap which maps addresses to the value stored at that place in memory.

Unlike MARS, I've made the design decision to use a hashmap. This allows for us to write anywhere in memory without being concerned with writing out of bounds (4 MB). The downside is that this has a higher overhead.
