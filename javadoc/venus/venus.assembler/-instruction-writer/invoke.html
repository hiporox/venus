invoke

operator fun invoke(prog: Program, iform: InstructionFormat, args: List<String>): Unit

Writes the instruction to the given program.

This describes how to write a particular instruction (like add), but not how to write its arguments (like add x0 x1 x2). That is, all the work for its arguments must be done by concrete classes.

prog - the program to add the instruction to iform - the required format of the instruction args - the arguments for this instruction abstract operator fun invoke(prog: Program, inst: Instruction, args: List<String>): Unit
